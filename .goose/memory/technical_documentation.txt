# html-editor react wysiwyg document-editor technical-reference
HTML Editor Library Project Structure:
- Monorepo with @prabhath-tharaka/html-editor (library) and demo-app (demo)
- Core components: HtmlEditor, EditorToolbar, PageView, PageManager
- Context-based state management with DocumentProvider
- Page sizing at 96 DPI (A4, Letter, Legal formats)
- Continuous scrolling with manual page breaks
- Rich text formatting capabilities
- Image handling with resize capabilities
